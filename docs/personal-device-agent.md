# EleMate Product Direction

## 한 줄 정의

EleMate는 `내 컴퓨터를 내 개인 에이전트로 바꾸는 설치형 앱`이다.

## 목표 사용자

- 개발을 모르는 일반 사용자
- 자기 장비를 원격으로 부리고 싶은 사용자
- 자기 데이터와 파일을 자기 장비 안에 두고 싶은 사용자

## 제품 원칙

1. 사용자는 `장비 연결`만 이해하면 된다.
2. 기술 인프라는 가능한 한 숨긴다.
3. 메인 경험은 채팅 하나로 시작한다.
4. 위험한 작업만 사람이 승인한다.
5. 실패와 오프라인 상태는 자연어로 설명한다.

## 사용자 경험

### 소유자

1. 앱 설치
2. AI 연결
3. 원격 연결 켜기
4. 내 폴더 선택
5. 휴대폰 링크 복사

### 휴대폰 사용자

1. 링크 열기
2. 자기 전용 대화창 접속
3. 채팅으로 지시
4. 필요 시 승인

## 사용자에게 보여주지 않을 것

- API
- localhost
- uvicorn
- serve
- tailnet
- runtime mode
- workspace root
- browser channel

## 기술 구조

### 각 설치본이 자체적으로 가지는 것

1. Electron 데스크탑 앱
2. 로컬 API
3. 로컬 웹 UI
4. 로컬 파일/셸/브라우저 실행기
5. 사용자 본인 GPT 계정 연결
6. 사용자 본인 Tailscale 연결

### 중앙 서버에서 하지 않는 것

1. 공용 사용자 데이터 저장
2. 공용 작업 실행
3. 공용 파일 저장소 중심 구조

## 지금 구현한 방향

1. 데스크탑 앱에서 장비 연결 상태를 안내
2. ChatGPT 로그인 기반 Codex 연결 사용
3. Tailscale 기반 개인 포털 자동 생성
4. 자유 채팅 중심 UX
5. macOS launchd 기반 상시 실행 스크립트 추가

## 아직 더 필요한 것

1. 첫 실행 마법사 더 단순화
2. 모바일 전용 레이아웃 다듬기
3. OS 전역 데스크탑 제어 확장
4. 설치 패키지 완전 독립형화
5. 오류 복구 자동화

## OpenClaw 대비 우리가 가져갈 차별점

1. 더 단순한 첫 연결
2. 개인 장비 중심 제품 설계
3. 채팅 우선 UX
4. 비개발자 친화 문구와 흐름
5. 각자 자기 계정과 자기 장비를 쓰는 구조
