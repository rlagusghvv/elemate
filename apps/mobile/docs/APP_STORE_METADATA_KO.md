# App Store metadata draft

## App name

옷날

## Subtitle

날씨에 맞는 옷장 추천

## Promotional text

아침마다 날씨를 확인하지 않아도, 내 옷장 기준으로 오늘 입을 조합을 빠르게 추천합니다.

## Description

옷날은 오늘 날씨와 내 옷장을 기준으로 아침 옷차림을 추천하는 앱입니다.

처음에 상의, 하의, 신발, 아우터를 한 번만 등록해두면 현재 위치의 기온, 체감 온도, 비 예보, 바람, 일교차를 반영해 오늘 입기 좋은 조합을 보여줍니다.

상품명을 검색해 실제 상품 정보를 불러오고, 내 옷장에 추가할 수 있습니다. 추위를 많이 타는지, 이동 시간이 긴지, 선호하는 스타일이 있는지도 반영해 추천을 조정합니다.

주요 기능:

- 현재 위치 기반 날씨 확인
- 체감 온도와 일교차를 반영한 옷차림 추천
- 내 옷장 아이템 등록 및 선택
- 실제 상품 검색 기반 옷장 추가
- 추위/더위 체감, 이동 방식, 스타일 태그 반영
